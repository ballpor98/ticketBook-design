# Attendees collection

## Data representation

```javascript
    {
        "id": string,
        "createdDt": datetime
    }
```

## Properties

| Properties ||
|-|-|
| id | **UUID4** |
| createdDt  | **datetime** |

## Example

```javascript
    {
        "id": "B8D802CF-DD8F-4E61-B15C-9E6C5844CCBB",
        "createdDt": "2021-01-18T23:59:59+07:00"
    }
```
